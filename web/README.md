# Веб-интерфейс wb_fin_reports

## Запуск API
```bash
uvicorn web.server:app --reload
```

## Просмотр отчётов
1. Запустите сервер (см. выше).
2. Откройте файл `web/index.html` в браузере (двойной клик или `file:///` URL).
3. Страница автоматически выполнит `fetch("/rrd")` и отобразит последние 100 записей.
