# Деплой wb-fin_reports на Render

1. Создайте новый **Web Service** на [render.com](https://render.com).
2. Подключите ваш репозиторий GitHub с проектом `wb-fin_reports`.
3. Render автоматически определит настройки благодаря `render.yaml` и `Procfile`.
4. Подтвердите параметры и запустите деплой — он выполнится автоматически и при следующих пушах.
